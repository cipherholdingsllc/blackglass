# Tokens

Blackglass tokens are public-safe design constraints.

## Color

| Token | Role |
|---|---|
| `color.bg.blackglass` | root black surface |
| `color.bg.obsidian` | primary background |
| `color.surface.graphite` | quiet panel |
| `color.surface.glass` | functional glass |
| `color.surface.raised` | prominent surface |
| `color.text.primary` | ivory text |
| `color.text.muted` | subdued copy |
| `color.signal.cyan` | active signal |
| `color.signal.green` | verified |
| `color.signal.amber` | decision |
| `color.signal.red` | boundary |
| `color.signal.purple` | abstract |
| `color.signal.slate` | deferred |

## Radius

| Token | Role |
|---|---|
| `radius.card` | cards and exhibit surfaces |
| `radius.pill` | status chips |
| `radius.panel` | large panels |

## Shadow

| Token | Role |
|---|---|
| `shadow.whisper` | quiet depth |
| `shadow.present` | active panel |
| `shadow.prominent` | hero and decision focus |

## Motion

| Token | Role |
|---|---|
| `motion.fast` | press and hover response |
| `motion.medium` | reveal and focus transitions |
| `motion.slow` | hero-level ambient shifts |

## Glass

| Token | Role |
|---|---|
| `glass.whisper` | quiet layering |
| `glass.present` | normal exhibit surface |
| `glass.prominent` | focused decision surface |
