# Visual Language

Blackglass is premium, dark, precise, and restrained. It should feel like a luxury instrument panel for complex context, not a nightclub dashboard.

## Foundation

- Deep obsidian background
- Graphite glass surfaces
- Ivory text instead of sterile white-only text
- Crisp hairline borders
- Restrained glow
- Semantic accents
- Cinematic depth
- Dense but readable spacing

## Accent Language

| Accent | Meaning |
|---|---|
| Cyan | Active signal or focus |
| Amber | Decision or waiting |
| Green | Verified or stable |
| Red | Boundary or blocked |
| Purple | Abstract or system-level |
| Slate | Deferred or quiet |

## Surface Rules

Glass surfaces should create hierarchy, focus, or boundary. If a layer is only decorative, remove it.

## Typography

Use a clean UI sans-serif for primary interface text and a monospaced face for status labels, counters, and structured metadata.

## Rejections

- rainbow neon
- cheap blur everywhere
- decorative blobs
- fake live status
- generic SaaS card grids
- cyberpunk cosplay
- inaccessible contrast
