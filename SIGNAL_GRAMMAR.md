# Signal Grammar

Signal grammar makes status legible across dense interfaces.

## Semantic Colors

| Color | Meaning |
|---|---|
| Cyan | Active signal, current focus, live attention |
| Amber | Decision, wait state, review needed |
| Green | Verified, stable, complete |
| Red | Boundary, blocked, do not cross |
| Purple | Abstract, system-level, model-like structure |
| Slate | Deferred, quiet, low-priority |

## Rules

- Never use color alone.
- Pair color with labels, position, and shape.
- Keep accent counts low per surface.
- Prefer semantic consistency over novelty.
- Do not make every metric glow.

## Status Language

Use calm words:

- Active
- Waiting
- Verified
- Blocked
- Deferred
- Review

Avoid fake urgency and invented live activity.
