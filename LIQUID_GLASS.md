# Liquid Glass

Liquid glass is a functional material rule, not a visual gimmick.

## Core Rule

Glass must earn its existence.

It can be used for:

- focus
- decision surfaces
- boundary states
- layered context
- active controls
- modal emphasis

It should not be used for:

- random decoration
- visual noise
- low-contrast text containers
- glitter effects
- making every panel look expensive without adding meaning

## Intensity Tiers

| Tier | Use |
|---|---|
| Whisper | background separation, quiet depth |
| Present | primary cards, exhibit surfaces |
| Prominent | decision gates, active controls, focused state |

## Motion

Glass motion should explain state: press, focus, reveal, save, or transition. Respect reduced-motion preferences and keep motion short.

## Accessibility

Glass must preserve text contrast. Never use blur as the only signal for disabled, active, or blocked states.
