# Public Boundary

Blackglass is a public visual lab. It uses synthetic examples and public-safe language only.

## Allowed

- public design principles
- generic interface patterns
- mock demo data
- synthetic examples
- sanitized screenshots
- general token names
- generic timeline, status, and decision examples
- Blackglass brand concepts
- public-safe component demos

## Forbidden Examples

The following terms and concepts are prohibited outside this file and `REDACTION_REPORT.md`:

- 2OPMD
- 2ndOpinionMD
- CipherOS
- OGrE
- Ollama
- PubMed
- patient
- diagnosis
- treatment
- HIPAA
- clinical
- medical advice
- backend contract
- private schema
- funding
- A16Z
- YC
- Claude prompt
- receipt
- private repo path
- collaborator names
- private founder story

## Publishing Rule

If a draft includes one of the prohibited examples, remove it, generalize it, or keep it in private notes. Do not publish it as Blackglass content.
