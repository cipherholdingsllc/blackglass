# Mobile Screen Shell

## Purpose
Safe-area layout, dark base, compact context, and thumb-zone action placement.

## States
- loading
- empty
- success
- blocked

## Invariant
The primary object and primary action are clear within three seconds.

## Accessibility
Uses semantic regions, readable contrast, and stable action placement.

## Do-not-use-when (Avoid)
Do not use a web dashboard grid as the first mobile structure.
