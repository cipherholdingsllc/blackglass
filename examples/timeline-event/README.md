# Timeline Event

## Purpose
A sequence item with source badge, time, status dots, and detail affordance.

## States
- new
- reviewed
- needs context
- archived

## Invariant
Every item shows when it happened and where its context came from.

## Accessibility
Status dots are paired with labels and left-edge structure.

## Do-not-use-when (Avoid)
Do not use for unordered items with no time relationship.
