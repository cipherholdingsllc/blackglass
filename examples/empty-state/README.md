# Empty State

## Purpose
A calm zero-state with one direct next action.

## States
- empty
- first use
- offline
- blocked

## Invariant
The state explains what is true now and gives exactly one next step.

## Accessibility
No fake urgency, no tiny dismiss targets, and no motion requirement.

## Do-not-use-when (Avoid)
Do not shame the user or imply missing activity.
