# Capture Control

## Purpose
A 72px prominent input control for high-value capture moments.

## States
- idle
- active
- paused
- review
- disabled

## Invariant
The control never drops below 72px and every state has a plain label.

## Accessibility
State is conveyed through label, icon shape, color, and size.

## Do-not-use-when (Avoid)
Do not hide important capture states behind animation alone.
