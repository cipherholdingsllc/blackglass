# Mobile Action Button

## Purpose
Touch-sized command control for primary, secondary, and quiet actions.

## States
- default
- pressed
- disabled
- loading

## Invariant
Primary actions are at least 56px tall and stay in the reachable lower screen.

## Accessibility
Uses native button semantics, visible focus, and text labels.

## Do-not-use-when (Avoid)
Do not use tiny icon-only controls for critical actions.
