# Screen Gallery

The v0.3 site includes real screenshot-ready capture cards. The files in `assets/screenshots/` should be captured from the rendered site after final visual review.

## Required Captures

| Capture | Purpose |
|---|---|
| Hero Command Deck | first impression |
| Exhibit Grid | proof of breadth |
| Glass Gate Card | decision state |
| Signal Grammar | semantic color system |
| Timeline Rail | sequence pattern |
| Trust Surface | source and confidence pattern |
| Boundary Panel | calm blocked state |
| Capture Cards | README and launch previews |

## Rules

- Capture actual rendered pages.
- Do not mock screenshots that the site cannot reproduce.
- Use synthetic content only.
- Keep images crisp and inspection-friendly.
